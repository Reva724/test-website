Command List

Match Commands

Command Name Description Usage
live Displays information about a live match !live [map|maps]
!matches Displays information about team matches !matches <firstteam> [secondteam]
!next Displays information about the next match !next
!schedule Displays schedule for current stage week !schedule
!standings Displays league and stage standings
for the current OWL season		 !standings

News Commands

Command Name Description Usage
!news Displays recent Overwatch League news !news
!vods Displays recent Overwatch League videos !vods

Player Commands

Command Name Description Usage
!compare Compares overall league stats and
hero specific stats between two players		 !compare <first> <second>
!player Shows info about a player !player <playername> [accounts|heroes] [expand]
!player <playername> [hero] <heroname>
!players Lists all Overwatch League players, and
players with time on a specific hero		 !players [hero]

Team Commands

Command Name Description Usage
!team Displays information about a
specific Overwatch League team		 !team <name> [accounts|schedule]
!teamcords Displays Discord servers for Overwatch League teams !teamcords
!teams Lists all Overwatch League teams !teams [division]

Utility Commands

Command Name Description Usage
!help Lists available commands !help [command]
!info Displays Pachimari information !info
!ping Pings the bot to check timing !ping
