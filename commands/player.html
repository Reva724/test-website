Command Name Description Usage
!player Shows info about a player !player <playername> [accounts|heroes|hero] [expand|hero]

Player Info

Look up specific information about your favorite Overwatch League players! You can view all of their top heroes and stats.












Player Accounts

See an Overwatch League player's account list with the optional account argument. Players can be on various platforms including Twitch, YouTube, Discord, Twitter, Instagram, Facebook, Reddit, and Weibo.










Player Heroes

Access an Overwatch League player's hero list with the optional heroes argument.








You can also see an expanded list by adding the optional expand argument (ex !player geguri heroes expand











Player Hero Stats

Retrieve an Overwatch League player's hero specific stats using !player <playername> hero <heroname>
